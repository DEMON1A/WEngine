WEngine

About WEngine

WEngine is a web framework written in python that allows you to create your own web-based applications using an advanced template engine and a fast server with your own option. WEngine is almost a user controlled application that allows the user to use his own things in everything that includes security protections WEngine offers on all applications you're creating.

Get Started

Speed

WEngine is written with a pure python code and multi threaded tasks that makes it much faster than other servers making it possible to manage everything on a shorter time than other servers.

Engine

In WEngine. It uses it's own template engine that makes it easy for developers to chain the front-end jobs with the back-end ones by managing the front-end content from the HTML template

User Friendly

WEngine is a user friendly application. it allows you to manage everything inside your application from a simple options that doesn't require more than a string edit on the settings code.

Cross Platform

WEngine works on multiple platforms like Windows, Mac, Linux. it's still under testing because it's on beta. but at the time of writting this. WEngine has been tested on Windows and Linux and it's working fine on both of them.
